In order to run the demo jupyter notebook file 'notebook_intro.ipynb'
you need to do the following (assuming you have installed a jupyter
notebook - tensorflow environment):


1. Start a shell (linux/macos) or a command window (windows)
and activate your tensorflow environment

2. Start jupyter notebook

e.g.

..> jupyter notebook

3. Find the folder where you saved the two files
'notebook_intro.ipynb'
'auto.png'

4. Click on the file 'notebook_intro.ipynb'
